# 统计卡片图标替换指南

## 当前效果

统计卡片现在使用占位图片，并具有以下特性：

- **图标位置**：卡片中央（桌面端）/ 左侧（移动端）
- **悬停动画**：
  - 卡片整体上浮
  - 图标容器放大并轻微旋转
  - 图标本身反向旋转
  - 金额数字放大
  - 阴影增强

## 如何替换为您自己的图片

### 1. 准备图片

将您的三张图片放入 `src/assets` 目录：

```
src/assets/
  ├── income-icon.png    # 收入图标
  ├── expense-icon.png   # 支出图标
  └── balance-icon.png   # 结余图标
```

**推荐图片规格：**
- 尺寸：64x64 像素或 80x80 像素
- 格式：PNG（支持透明背景）或 JPG
- 背景：透明或白色

### 2. 修改代码

打开 `src/components/StatsPanel.vue`，找到模板部分，替换图片路径：

```vue
<!-- 收入卡片 -->
<div class="stat-icon-wrapper">
  <img 
    src="@/assets/income-icon.png" 
    alt="收入图标" 
    class="stat-icon-img"
  />
</div>

<!-- 支出卡片 -->
<div class="stat-icon-wrapper">
  <img 
    src="@/assets/expense-icon.png" 
    alt="支出图标" 
    class="stat-icon-img"
  />
</div>

<!-- 结余卡片 -->
<div class="stat-icon-wrapper">
  <img 
    src="@/assets/balance-icon.png" 
    alt="结余图标" 
    class="stat-icon-img"
  />
</div>
```

### 3. 调整样式（可选）

如果需要调整图标大小或样式，修改 CSS 部分：

```css
.stat-icon-wrapper {
  width: 80px;      /* 容器宽度 */
  height: 80px;     /* 容器高度 */
  /* ... 其他样式 */
}

.stat-icon-img {
  width: 64px;      /* 图片宽度 */
  height: 64px;     /* 图片高度 */
  /* ... 其他样式 */
}
```

### 4. 调整边框颜色

每个卡片的图标容器有对应的边框颜色：

```css
.income-card .stat-icon-wrapper {
  border: 2px solid #67C23A;  /* 绿色 */
}

.expense-card .stat-icon-wrapper {
  border: 2px solid #F56C6C;  /* 红色 */
}

.balance-card .stat-icon-wrapper {
  border: 2px solid #409EFF;  /* 蓝色 */
}
```

## 动画效果说明

当前实现的悬停动画包括：

1. **卡片整体**：向上移动 4px，阴影增强
2. **图标容器**：放大 1.1 倍，顺时针旋转 5 度
3. **图标图片**：逆时针旋转 5 度（与容器形成对比效果）
4. **金额数字**：放大 1.05 倍
5. **标签文字**：颜色加深

所有动画都有 0.3 秒的过渡效果，让交互更流畅。

## 响应式设计

- **桌面端（>768px）**：图标在卡片中央，内容居中显示
- **移动端（≤768px）**：图标在左侧，内容左对齐
- **小屏幕（≤480px）**：图标和文字尺寸自动缩小

## 在线占位图片说明

当前使用的占位图片来自 via.placeholder.com：
- 收入：绿色背景，白色 "+" 符号
- 支出：红色背景，白色 "-" 符号
- 结余：蓝色背景，白色 "$" 符号

这些图片仅用于演示，替换为您的图片后效果会更好。
