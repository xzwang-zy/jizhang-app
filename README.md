# 收支记账单

一款轻量化收支记账单页面应用工具，采用 Vue3 + Vite + Element Plus 技术栈开发，使用 localStorage 进行本地数据存储，无需后端服务支持。支持多账本管理，数据按账本隔离存储。

## 功能说明

### 1. 收支录入表单
- 金额输入框（支持两位小数，必须为正数）
- 收支类型选择器（收入/支出）
- 分类下拉菜单（餐饮、交通、购物、工资、娱乐、医疗、房租、学习、其他）
- 备注文本框（最多200字符）
- 日期选择器（默认当前日期）
- 完整表单验证（必填项、金额格式校验）
- 保存记录/重置功能

### 2. 账单列表
- 按时间倒序展示所有账单记录
- 显示完整信息：日期、金额、收支类型、分类、备注
- 分页功能（每页8条记录）
- 编辑功能（弹窗表单，支持修改所有字段）
- 删除功能（带二次确认）
- 新增功能（列表顶部新增按钮，弹窗表单录入）
- 空状态提示

### 3. 筛选功能
- 月份筛选（下拉选择）
- 收支类型筛选（收入/支出/全部）
- 分类筛选（与类型联动，支出：餐饮、交通、购物、娱乐、医疗、房租、学习、其他；收入：工资）
- 统计面板快捷筛选（点击收入/支出卡片快速筛选）
- 条件组合使用，结果实时更新
- 清除筛选快捷按钮

### 4. 统计面板
- 本月总收入（绿色数字）
- 本月总支出（红色数字）
- 本月结余（根据正负显示不同颜色）
- 数据根据筛选条件动态更新
- 卡片悬停动画效果（缩放、旋转、阴影增强）
- 支持点击快捷筛选对应类型数据
- 自定义图标支持（可替换为主题图片）

### 5. 可视化图表
- 月度收支饼图（ECharts）
- 分别展示收入分类占比和支出分类占比
- 支持交互（悬停显示具体数值和百分比）
- 无数据友好提示

### 6. Excel导出功能
- 导出全部账单/导出当前筛选账单两种模式
- Excel表头：日期、收支类型、分类、金额、备注
- 文件名自动携带当前月份（格式：收支账单_YYYY-MM.xlsx）
- 导出成功/失败提示弹窗
- 导出按钮位于筛选栏右侧，与清除筛选按钮并排

### 7. 多账本管理
- 默认内置日常、旅游、工作三类账本
- 支持新增自定义账本（点击"新增账本"显示输入框，输入名称后点击其他区域自动保存）
- 支持删除自定义账本（每个账本条目右侧显示删除按钮，点击弹出确认对话框）
- 账本名称双击编辑（双击名称转为输入框，点击其他区域自动保存）
- 默认账本名称不可修改、不可删除
- 账本切换下拉选择，数据按账本隔离存储
- 月度预算绑定对应账本，切换账本自动切换预算统计
- 当前选中账本存入本地存储，刷新页面自动恢复
- 筛选、饼图统计、导出Excel仅作用于当前选中账本

### 8. 图标配置
- 点击页面顶部"图标配置"按钮打开配置面板
- 支持上传自定义图标（JPG、PNG、SVG格式，最大2MB）
- 支持调整每个图标的尺寸大小（32px-120px）
- 支持通过拖拽或坐标输入调整图标位置
- 实时预览调整效果
- 配置参数（含自定义图标）保存至本地存储，刷新页面自动恢复
- 支持一键重置为默认配置
- 移除已上传图标后自动恢复默认图标

## 环境要求

- Node.js: >= 14.0.0
- npm: >= 6.0.0

## 安装与运行

```bash
# 安装依赖
npm install

# 开发模式运行
npm run dev

# 构建生产版本
npm run build

# 预览生产版本
npm run preview
```

## 技术栈

- Vue 3.4+
- Vite 5.2+
- Element Plus 2.5+
- ECharts 5.5+
- xlsx 0.18+（Excel导出）

## 设计特点

- **简约浅色主题**：清爽的视觉体验
- **圆角卡片设计**：统一8px圆角
- **柔和蓝色主色调**：#409EFF
- **响应式布局**：适配桌面端和移动端
- **交互反馈**：过渡动画和操作提示

## 数据存储

- 使用 localStorage 存储所有账单数据
- 数据结构包含：id、amount、type、category、date、note、createdAt
- 支持数据的增删查改操作
- 异常处理机制确保数据安全

## 浏览器兼容性

支持现代浏览器：Chrome、Firefox、Safari、Edge
